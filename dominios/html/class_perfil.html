Detailed Description

Classe responsável por representar o perfil de investimento de um cliente. Oferece três tipos de perfil pré-definidos: CONSERVADOR, MODERADO e AGRESSIVO. Permite armazenar, recuperar e converter para string o perfil do investidor.
