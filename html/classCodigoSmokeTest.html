Detailed Description

verifica a estabilidade e funcionalidade básica do dominio codigo em uma compilação

Os testes abrangem:

  • Codigo válido.
  • Codigo com comprimento maior que o permitido.
  • Código com comprimento menor que o permitido
  • Código com caracteres inválidos.

Lança exceção caso o codigo informado seja inválido.
