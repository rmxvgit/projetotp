\doxysubsection{Detailed Description}
Entidade responsável por registrar e gerenciar as operações financeiras de um investidor. 

A entidade \doxylink{classAccount}{Account} representa uma conta de usuário no sistema, armazenando informações essenciais como CPF, nome e senha, todos validados conforme suas regras de domínio.

Regras de formato\+:
\begin{DoxyItemize}
\item O CPF deve estar em formato válido.
\item O nome deve ser válido
\item A senha deve ser válida 
\end{DoxyItemize}
