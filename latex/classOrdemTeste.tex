Classe responsável por realizar testes unitários na classe \doxylink{classOrdem}{Ordem}. 

Esta classe executa cenários de teste para verificar se a classe \doxylink{classOrdem}{Ordem} está funcionando corretamente. Ela define métodos para configurar o ambiente de teste, executar casos de sucesso e falha, e limpar os dados.
