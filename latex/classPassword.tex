\doxysubsection{Detailed Description}
Sequência secreta de caracteres utilizada para autenticar um usuário e conceder-\/lhe acesso a recursos ou funções protegidas. 

Regras de formato\+:


\begin{DoxyItemize}
\item Seis caracteres.
\item Cada caracter pode ser dígito (0 – 9), letra (A – Z ou a – z), \#, \$, \% ou \&.
\item Não há caracter duplicado.
\item Existe pelo menos um dígito (0 – 9).
\item Existe pelo menos uma letra maiúscula (A – Z).
\item Existe pelo menos uma letra minúscula (a – z).
\item Existe pelo menos um caracter especial (\#, \$, \% ou \&). 
\end{DoxyItemize}
